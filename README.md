# Better Tele-Task

**Better Tele-Task** (BTT) is a [browser extension](https://github.com/vieledatengutedaten/better-teletask-extension) created by HPI students who love Tele-Task but are sometimes also annoyed by it. If you feel the same, this extension might be right for you.

This is the server component that creates the subtitles and makes them available to the extension via API.
These subtitles are generated using publicly available deep-learning-based automatic speech recognition models.
